## Description

This is a project made during NestJS Zero to Hero course from Udemy.
I'm using [Nest](https://github.com/nestjs/nest) framework TypeScript starter repository to bootstrap the project.
Check them out. They're awesome.

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
